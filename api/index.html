Welcome to the MML REST API 
We have the following endpoints

/thermostat - Relays thermostat JSON
/weather/[zipcode] - Gets weather for the specified zip code (simulated MMLAPI restricted to zipcode=32608)

/torrent/[magnet link] - Downloads a torrent using the magnet link given

/music_resync/ - Runs ansible playbook to resync all music Pis

/sign - Lists sign endpoints
/sign/allon - Turns the sign on
/sign/alloff - Turns the sign off
/sign/onebyone - Turns the sign on one by one

End of documentation
